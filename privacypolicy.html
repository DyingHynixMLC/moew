moew Privacy Policy

1. Introduction
This Privacy Policy describes how your data is collected, used, and protected when using the Bot.

2. Data Collected
The Bot does not collect any personally identifiable information (PII) or store any user data related to your interactions. The Bot only processes the commands you send to fetch random pictures of cats and foxes from external APIs.

The API calls made to The Cat API and RandomFox API may log basic usage data, such as IP addresses or user-agent information, but this data is outside the control of the Bot developer.

3. How We Use Data
- The Bot processes user commands to retrieve cat and fox images. No additional user data is stored or used for purposes other than responding to the user's request.

4. Data Sharing
- The Bot does not share or sell any personal data.
- The external APIs (The Cat API and RandomFox API) may collect and use data in accordance with their own privacy policies.

5. Data Security
While we take precautions to ensure the security of user data, no method of transmission over the internet or method of electronic storage is 100% secure. We cannot guarantee absolute security.

6. Third-Party Services
The Bot uses third-party services to fetch random cat and fox images. The use of these APIs is governed by their respective privacy policies:
- The Cat API Privacy Policy
- RandomFox API Privacy Policy

7. User Rights
Since the Bot does not collect personal information, there is no personal data stored or shared by the Bot. However, if you have concerns about data collected by the external APIs, please refer to their privacy policies.

8. Changes to the Privacy Policy
We may update this Privacy Policy from time to time. Any changes will be posted here with an updated date.

9. Contact Information
For any questions or concerns regarding this Privacy Policy, please contact the bot developer on GitHub.
